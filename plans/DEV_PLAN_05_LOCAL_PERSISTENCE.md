# PopSearch MVP 개발 계획 - 5단계 로컬 저장(엔진/최근 검색)

## 목표
- 마지막 엔진과 최근 검색을 재실행 후에도 유지한다.

## 핵심 작업
- 저장소 계층 구현(예: `shared_preferences`)
- 마지막 사용 엔진 저장/복원
- 최근 검색 정책 정의:
  - 최대 20개
  - 중복 시 최신 우선 정렬
- 최근 검색 탭 동작 일관화:
  - 즉시 검색 또는 입력 채우기 중 하나로 고정

## 산출물
- PreferencesRepository
- HistoryRepository

## 완료 기준(DoD)
- 앱 재시작 시 엔진 복원
- 최근 검색 저장/표시/재사용/삭제가 정책대로 동작
