# PopSearch MVP 개발 계획 - 4단계 외부 핸드오프 안정화

## 목표
- 검색 실행 후 외부 브라우저/앱 핸드오프를 안정적으로 제공한다.

## 핵심 작업
- 실행 서비스(launch abstraction) 구현
- 시스템 기본 브라우저 존중 정책 적용
- YouTube 처리:
  - 유니버설 링크 우선
  - 실패 시 웹 폴백
- 예외 처리:
  - 빈 쿼리 차단
  - 실행 실패 시 사용자 안내

## 산출물
- SearchExecutionService
- 핸드오프 관련 테스트(모킹)

## 완료 기준(DoD)
- 엔진별 결과 페이지/앱 이동이 안정적으로 동작
- YouTube 미설치 환경에서도 검색 실패 없음
