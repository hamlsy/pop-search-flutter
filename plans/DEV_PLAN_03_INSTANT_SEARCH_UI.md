# PopSearch MVP 개발 계획 - 3단계 즉시 검색 UI

## 목표
- 앱 진입 즉시 검색 가능한 메인 UI를 완성한다.

## 핵심 작업
- 필수 UI 구성:
  - 엔진 선택 버튼/칩
  - 검색 입력 필드
  - Clear(X)
  - Go 버튼
  - 최근 검색 리스트 영역
- 화면 진입 시 자동 포커스 + 키보드 노출
- Enter/Go 동등 검색 트리거 연결
- Soft Minimal 스타일 반영:
  - 라운드 카드
  - 부드러운 그림자
  - 여백 중심 배치

## 산출물
- SearchScreen v1

## 완료 기준(DoD)
- 앱 실행 직후 바로 입력 가능
- Enter/Go 모두 동일 경로로 검색 실행
