# PopSearch MVP 개발 계획 - 9단계 테스트 및 릴리즈 게이트

## 목표
- 기능/정책/품질을 릴리즈 가능한 수준으로 검증한다.

## 핵심 작업
- 테스트 보강:
  - URL 빌더 단위 테스트
  - 저장소(엔진/히스토리/테마) 단위 테스트
  - 검색 플로우 위젯 테스트
- 분석/빌드 검증 루틴 정리
- Flow A~E 기반 수동 QA 시나리오 작성

## 산출물

- QA 체크리스트
