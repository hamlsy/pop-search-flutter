# PopSearch MVP 개발 계획 - 10단계 스토어 출시 준비

## 목표
- App Store/Play Store 제출 가능한 상태로 마무리한다.

## 핵심 작업
- iOS:
  - 유료 가격 티어 설정(USD 1 인접)
  - 광고 SDK 미포함 최종 검증
- Android:
  - Data Safety/개인정보 고지 반영
  - 광고 동작 정책 재검증(비방해/비기만)
- 공통:
  - 개인정보 처리 문구 정리(로컬 저장 항목, Android 광고 SDK)
  - 스토어 설명/스크린샷/비제휴 고지 정합성 점검

## 산출물
- 스토어 제출 체크리스트 완료본
- RC 빌드

## 완료 기준(DoD)
- PRD MVP Definition of Done 항목 전부 충족
- 제출 메타데이터와 실제 앱 동작 일치
