# PopSearch 스토어 설명 초안

## 한줄 설명
빠르게 엔진을 고르고 바로 검색하는 미니멀 검색 런처.

## 주요 기능
- Google, YouTube, Naver, Daum, Yahoo 즉시 검색
- 최근 검색 저장/재사용/삭제
- 테마 모드(System/Light/Dark) 및 재실행 유지
- Android: Settings 하단 비방해 배너 1개

## 개인정보/정책 요약
- 계정 로그인 없음
- 검색 기록/설정은 기기 로컬에만 저장
- iOS 빌드는 광고 SDK 미포함
- Android 빌드는 Settings 하단 배너 광고 SDK 포함
